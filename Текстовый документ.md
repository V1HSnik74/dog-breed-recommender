Техническое задание на проект: Сервис подбора пород собак для будущих владельцев с целью рекомендации подходящей породы по образу жизни и условиям содержания
1. Цель проекта: Создание сервиса для подборки породы собаки. Система помогает пользователям определиться с выбором питомца, предоставляя несколько вариантов по введенным пользователем данным.
2. Роли пользователей: Гость
3. Модели данных:
    Модель 1: порода собаки, содержит поля название породы, минимальный и макисмальный вес (из которых вычисляется средний вес), насколько сильно собака линяет,  насколько тяжелый уход требует, длина шерсти, насколько энергичная, насколько обучаемая, насколько хорошо ладит с детьми и другими собаками
    Модель 2: опросник, содержит поля тип жилья, наличие детей, уровень активности, предпочитаемый вес собаки (т.е. размер), наличие аллергии, наличие других собак, наличие времени для грумминга
    Модель 3: пока в разработке
4. Ключевой функционал
    Анкета подбора: пользователь заполняет анкету, чтобы получить персональные рекомендации по породам
    Каталог пород: пользователь просматривает карточки пород с характеристиками
    Алгоритм рекомендаций: пользователь получает топ-10 пород
5. Внешние интеграции
     Интеграция: использование внешнего API для получения каталога пород с собаками
