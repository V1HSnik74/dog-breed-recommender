Техническое задание на проект: Сервис подбора пород собак для будущих владельцев с целью рекомендации подходящей породы по образу жизни и условиям содержания
1. Цель проекта: Создание сервиса для подборки породы собаки. Система помогает пользователям определиться с выбором питомца, предоставляя несколько вариантов по введенным пользователем данным.
2. Роли пользователей: Гость, может заполнить форму и просмотреть 5 предложенных ему пород собак.
3. Модели данных:
    Модель 1: порода собаки (DogBreed), содержит поля название породы (name), минимальный(min_weight) и макисмальный(max_weight) вес (из которых вычисляется средний вес(avg_weight)), насколько сильно собака линяет(shedding), насколько тяжелый уход требует(grooming), насколько энергичная (energy), насколько обучаемая(trainability), насколько хорошо ладит с детьми(good_with_children) и другими собаками(good_with_other_dogs)
    Модель 2: опросник (Survey), содержит поля тип жилья(home_type), наличие детей(has_children), уровень активности(activity_level), предпочитаемый вес собаки (т.е. размер)(preferred_weight), наличие аллергии(has_allergy), наличие других собак(has_other_dogs), наличие времени для грумминга(has_time_for_grooming)
    Модель 3: рекомендации (Recommendation), содержит поля survey (внешний ключ, ссылается на Survey), top_5_dog_json (хранит информацию о топ-5 пород собак, подходящих пользователю)
4. Ключевой функционал
    Анкета подбора: пользователь заполняет анкету, чтобы получить персональные рекомендации по породам
    Каталог пород: пользователь просматривает карточки пород с характеристиками, видит процентное соответствие породы собаки
    Алгоритм рекомендаций: пользователь получает топ-5 пород
5. Внешние интеграции
     Интеграция: использование внешнего API для получения каталога пород с собаками (API-ninja dogs)
